## QA Testing (Sentinel)

This project uses Sentinel for NL spec-driven QA testing with Playwright Test Agents.

- **NL Specs**: `specs/` — natural language spec files that describe feature behavior
- **Generated Tests**: `tests/` — `.spec.ts` files generated by Playwright Planner/Generator
- **Config**: `sentinel.config.yaml`
- **Authoring adapters**: generated by `/qa:setup`; read `sentinel.config.yaml` for project facts and should not be hand-customized

### Commands

- `/qa:setup` — Initialize Sentinel (already done)
- `/qa:discover` — Scan app and author NL specs
- Drift detection via `/workflow:audit` (qa domain)
